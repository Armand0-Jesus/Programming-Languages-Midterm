Programming Languages Midterm Project

Smart Academic Task Planner

Una sola web personal con 7 programas ejecutables, organizada por modulos y paradigmas.

Python - Calculadora de promedio / GPA

Paradigma: Imperativo | Estado: Funcional minimo

Ingresa hasta 8 clases con nota y creditos. La nota puede ser A, B, C, D, F o porcentaje.

Resultado: listo para calcular.

Python - Conversor academico / utilitario

Paradigma: Imperativo | Estado: Funcional minimo

Selecciona una conversion, escribe el valor y presiona convertir.

Resultado: listo para convertir.

Java - Task Manager simple

Paradigma: Orientado a Objetos | Estado: Funcional minimo

Agrega tareas y eliminalas cuando ya no las necesites. Se guardan localmente.

Estado: listo

    Java - Calculadora de prioridad de tareas

    Paradigma: Orientado a Objetos | Estado: Funcional minimo

    Ingresa urgencia, dificultad y dias restantes para obtener prioridad sugerida.

    Resultado: listo para calcular.

    Tau Prolog - Recomendador basado en reglas

    Paradigma: Logico | Estado: Funcional minimo

    Marca las condiciones de hoy y genera una recomendacion de estudio basada en reglas.

    Resultado: listo para analizar.

    Elm - Budget / Expense Tracker simple

    Paradigma: Funcional | Estado: Funcional minimo

    Registra ingresos y gastos para ver tu balance total en tiempo real.

    Estado: cargando modulo Elm...

    CV / Resume Generator

    Requisito especial | Estado: Funcional minimo

    Carga cv.json, activa o desactiva entradas y descarga tu resume en PDF.

    Estado: cargando datos...
